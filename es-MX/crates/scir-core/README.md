scir-core

Descripción general
- Utilidades centrales para SciR (SciPy reconstruido para Rust): ayudantes de tolerancia numérica, utilidades de números complejos y una macro `assert_close!` reutilizable utilizada en todos los crates.

Aspectos destacados
- Comparación consistente de punto flotante a través de `assert_close!` para slices y arrays ndarray (reales y complejos).
- Bloques de construcción mínimos y probados en batalla compartidos por `scir-fft`, `scir-signal`, `scir-optimize` y otros.
