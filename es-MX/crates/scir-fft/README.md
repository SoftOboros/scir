scir-fft

Descripción general
- Utilidades FFT para SciR con un enfoque de paridad primero. Utiliza accesorios generados por SciPy para validar `fft`, `ifft`, `rfft` e `irfft` contra salidas de referencia.

Backends
- CPU: rustfft (complejos) y realfft (transformadas reales).

Enlaces
- Repositorio: https://github.com/SoftOboros/scir
- Documentación: https://docs.rs/scir-fft
