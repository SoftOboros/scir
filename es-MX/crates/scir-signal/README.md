scir-signal

Resumen
- Procesamiento de señales para SciR: diseño de filtros clásicos (Butterworth, Chebyshev I, Bessel), filtrado de secciones de segundo orden, `filtfilt` de fase cero y `resample_poly`.
- Paridad primero: validado contra SciPy mediante fixtures.
- GPU opcional: la ruta FIR puede enviarse automáticamente a CUDA cuando está habilitada.

Enlaces
- Repositorio: https://github.com/SoftOboros/scir
- Documentación: https://docs.rs/scir-signal
