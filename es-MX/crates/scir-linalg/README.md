scir-linalg

Resumen
- Componentes fundamentales de álgebra lineal para SciR. Rutas respaldadas por BLAS/LAPACK a través de `ndarray-linalg` y un backend `faer` puro de Rust planificado.

Estado
- Aún no publicado en crates.io; la superficie de la API está en desarrollo activo.
