scir-fft

Overview
- FFT utilities for SciR with a parity-first approach. Uses fixtures generated from SciPy to validate `fft`, `ifft`, `rfft`, and `irfft` against reference outputs.

Backends
- CPU: rustfft (complex) and realfft (real transforms).
